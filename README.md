# IMC.py
### Um script simples em Python desenvolvido para calcular o Índice de Massa Corporal (IMC) do usuário e indicar sua classificação de peso de acordo com a tabela padrão. O programa roda em loop, permitindo múltiplos cálculos até que o usuário decida encerrar 😀
